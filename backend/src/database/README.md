# Beat The Heat - Database Documentation

## Overview
This folder contains all database-related files for the AI-Integrated Smart Heat Index and Real-Time Health Advisory System.

## Folder Structure

### migrations/
Contains SQL migration files for creating and modifying database tables.

### seeds/
Sample data for development and testing.

### functions/
Custom database functions for calculations and data processing.

### triggers/
Database triggers for automated actions.

### policies/
Row Level Security (RLS) policies for Supabase.

### views/
Database views for complex queries.

### indexes/
Database indexes for performance optimization.

## Setup Instructions

1. Create a Supabase project
2. Run migrations in order (001, 002, 003, etc.)
3. Apply RLS policies
4. Create functions and triggers
5. (Optional) Load seed data for testing
